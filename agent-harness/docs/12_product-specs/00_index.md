# PRODUCT SPECS INDEX

> 기능 명세의 인덱스.
> `PROJECT_INTAKE`가 기획 입력을 모으는 문서라면, 이 폴더는 기능 단위 명세를 정리하는 위치다.

---

## Status Guide

- `draft`: 초안
- `reviewing`: 검토 중
- `approved`: 구현 가능
- `implemented`: 구현 반영됨
- `deprecated`: 더 이상 유지하지 않음

---

## Recommended Spec Structure

각 기능 명세는 최소한 아래를 포함한다.

1. 문제와 배경
2. 사용자 시나리오
3. 범위 포함 / 제외
4. 기능 요구사항
5. 비기능 요구사항
6. acceptance criteria
7. 오픈 질문
8. 관련 exec-plan 링크
