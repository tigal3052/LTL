# PC Environment

이 폴더는 Looting The Leviathan 작업에서 반복적으로 필요한 로컬 PC 전용 정보를 둡니다.

필수 용도:

- Godot 실행 파일 경로를 다시 찾지 않기
- PowerShell / WSL 차이로 인한 실행 이슈를 반복하지 않기
- Node, Python, Git 같은 런타임 버전과 경로를 빠르게 확인하기

작업 규칙:

- 로컬 절대 경로가 필요한 작업은 이 폴더를 먼저 확인합니다.
- 문서에 없는 경로를 새로 찾았다면, 작업을 끝내기 전에 여기 갱신합니다.
- 실행 계획과 완료 문서에서 참조한 환경 문서를 명시합니다.
