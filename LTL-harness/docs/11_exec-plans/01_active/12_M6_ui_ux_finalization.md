# M6. UI/UX 정리

## 목표

전투 판단을 방해하지 않는 선에서 LTL의 채굴 판타지와 보상 쾌감을 강화한다.

## 산출물

- 최종 HUD 배치
- tooltip/상태 아이콘
- reward presentation
- failure/retry flow
- accessibility pass

## QA 대상

- 텍스트와 UI 겹침 없음
- 색상만으로 파동을 구분하지 않도록 보조 형태/아이콘 제공
- 4핀 경고와 큐 상태가 즉시 읽힘

## 완료 기준

- 반복 플레이 중 필요한 정보가 1초 안에 식별된다.
- 수동 QA 체크리스트 전 항목 통과.
