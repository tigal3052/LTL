# M7. 내러티브 통합

## 목표

채집과 사냥의 금기, 주인공과 단장의 갈등을 게임 흐름에 얹되 전투 리듬을 끊지 않는다.

## 산출물

- intro narrative beat
- 첫 유물 획득 후 갈등 beat
- run failure/clear text
- terminology pass

## 구현 원칙

- 전투 중 긴 대사는 금지한다.
- 노드 선택, 보상, 런 종료 화면에 짧게 배치한다.
- `사냥`과 `채집`의 차이를 문구와 연출에서 분명히 한다.

## 완료 기준

- 튜토리얼 없이도 플레이어가 LTL의 금기를 이해한다.
- 서사가 replay/도메인 테스트에 영향을 주지 않는다.
