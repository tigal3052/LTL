# M3. 보상과 성장

## 목표

노드 클리어 후 보상 선택과 다음 빌드 변경이 재미를 이어가도록 구현한다.

## 산출물

- reward offer/selection flow
- rarity/weight table
- run-level growth state
- 보상 연출 최소 버전

## TDD 대상

- 보상 수량/가중치 재현성
- 선택한 보상이 백팩/런 상태에 반영
- 골드/경험치 산출 로그

## 완료 기준

- 보상 선택 후 다음 노드 전투에 영향이 나타난다.
- reward telemetry가 빌드 변화 추적에 충분하다.
