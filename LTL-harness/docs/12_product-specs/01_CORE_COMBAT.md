# Core Combat Specs

## 전투 정체성

전투는 적을 공격하는 장면이 아니라 레비아탄의 표면을 읽고, 파동에 맞는 에너지를 쏘아 마력장과 외피를 뚫는 채굴 행위입니다.

## 지형과 파동

- 전투 하단은 3행 x 10열 지형 그리드입니다.
- 각 셀은 현재 파동 색상과 비활성화 상태를 가진다.
- 파동 계열은 물리력(빨강), 마력(파랑), 부정(보라), 부패(초록) 4종입니다.
- 노드는 취약 파동 0~2개를 가진다. 취약 파동이 없으면 노멀 지형입니다.
- 노멀 지형은 어떤 파동도 보너스/미스매치 없이 기본 피해로 처리합니다.

## 내구도 구조

- 지형은 `마력장`과 `체력` 두 구간을 가진다.
- 마력장이 남아 있으면 체력 피해보다 마력장 피해를 우선 적용한다.
- 마력장이 0 이하가 되면 이후 유효 피해는 체력에 적용한다.
- 체력이 0 이하가 되면 노드 클리어입니다.

## 데미지 판정

| 조건 | 마력장 피해 | 체력 피해 | 설명 |
|---|---:|---:|---|
| 취약 파동 매칭 | 계열 기본 배율 x 1.5 | 계열 기본 배율 x 1.5 | 약점 공략 |
| 노멀 지형 | 계열 기본 배율 x 1.0 | 계열 기본 배율 x 1.0 | 안전 선택지 |
| 취약 파동 불일치 | 0.2배 | 0.5배 | 시간 낭비에 가까운 페널티 |

계열 기본 배율:

- 물리력: 마력장 0.5, 체력 1.2
- 마력: 마력장 1.5, 체력 0.75
- 부정: 마력장 0.75, 체력 0.75, 디버프 확장 후보
- 부패: 마력장 0, 체력 1.0, 지속 피해 확장 후보

## 입력과 발사

- hover는 항상 셀 1개만 선택해야 한다.
- 클릭 1회는 즉시 1발을 발사한다.
- 마우스 버튼 hold는 0.1초 간격으로 연속 발사한다.
- 발사는 에너지 큐의 맨 앞 항목 1개를 소비한다.
- 큐가 비어 있으면 발사 실패, 내구도 손실 1회, `shots_fired_empty_queue` 증가.
- 같은 타일은 유효 피격 후 0.5초 동안 비활성화된다.
- 비활성 타일에 발사하면 에너지는 소비하지만 판정은 무효, `shots_invalid_tile_cooldown` 증가.

## 4핀 실패 타이머

- 4핀은 시간 경과로만 소진된다.
- 기본 트리거는 25%, 50%, 75%, 100% 경과입니다.
- 방해 요소는 핀을 직접 제거하지 않는다.
- `clear`와 `time_over`가 같은 tick에 발생하면 `clear`를 우선한다.
