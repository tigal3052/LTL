# Progression & Meta Reward

## 보상 루프

노드 클리어 시 지형이 붕괴하고 균열에서 보상이 등장합니다. 보상은 전투 성공의 쾌감과 다음 빌드 계획을 동시에 제공해야 합니다.

## 드롭 수량

| 수량 | 확률 |
|---:|---:|
| 1개 | 25% |
| 2개 | 35% |
| 3개 | 20% |
| 4개 | 15% |
| 5개 | 5% |

프로토타입에서는 확률 자체보다 보상 후보가 현재 파동/노드 특성과 연결되는지 검증합니다.

## 보상 후보 가중치

- 클리어한 노드의 취약 파동과 같은 파동 태그 유물 가중치 증가
- 현재 백팩에서 부족한 역할(`repair`, `power`, `support`) 후보 최소 1개 보장 후보
- 전설/신화는 P4 이전에는 실제 밸런싱 대상이 아니라 데이터 구조 검증 대상으로 둔다

## 노드 진행

- 각 스테이지는 노멀 1개 고정 + 랜덤 지형/이벤트 4개 이상을 후보로 제시한다.
- 프로토타입 P4는 3스테이지 미니 런을 기준으로 한다.
- 정식 구현은 3~5스테이지, 스테이지당 5~10노드를 목표로 한다.

## 영구 성장

정식 구현 전까지 영구 성장은 로그와 보상 구조만 남기고 실제 강화 효과는 최소화합니다.

- 골드: 런 종료 후 유물 점수 환산, 장비/캐릭터/유물 해금 후보
- 경험치: 진행 단계 비례 지급, 패시브 스탯 트리 후보

## 테스트 대상

- 같은 seed의 보상 후보 재현성
- 드롭 수량 분포가 테이블을 따른다
- 노멀 노드가 항상 선택지에 포함된다
- 보상 선택 후 백팩/런 상태가 올바르게 갱신된다
